# 🍕 Pizza Sales SQL Analysis Project

## 📌 Project Overview

This project analyzes pizza sales data using SQL to find useful business insights.

## 🛠️ Tools Used

* MySQL
* SQL
* Canva

## 📊 Dataset

* orders
* order_details
* pizzas
* pizza_types

## 🔍 Analysis Performed

* Total number of orders
* Total revenue
* Most popular pizza size
* Top 5 ordered pizzas
* Category-wise sales
